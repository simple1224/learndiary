for循环

语法：
        for(初始化变量;循环条件;循环迭代)
    {
        循环语句
    }

while循环

语法：
        while(判断条件)
    {
        循环语句
    }

do...while循环

语法：do...while和while的区别是：前者会至少循环一次。先执行，后判条件 
        do
    {
        循环语句
    }
    while(判断条件)

退出循环break

在while,do...while,for循环中使用break中断循环，执行后面的程序

继续循环continue

continue的作用是跳出本次循环，而整体的循环继续

函数

函数的作用是：只写一次代码，可以实现重复利用。

语法：
        function  函数名( )
    {
        函数体;
    }

函数的调用：一个函数写好之后不会自动调用

  • 第一种方式是直接在script中调用，即直接写函数名()
  • 第二行方式是在HTML中调用，即利用click事件调用

    • 有参数的函数:参数可以有多个，用逗号隔开

    语法：
            function 函数名(参数1,参数2)
    {
        函数代码
    }

    返回值的参数：返回值可以是数字，字符串等多种形式
